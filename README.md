# expotech-ecogreen
Sistema de gerenciamento e controle de envios para o agronégocio desenvolvido usando Python e MySQL

# Fucionalidades 
* Gerenciamento de Fazendas: Cadastro, listagem, atualização e remoção de estoque
* Catálogo de Plantas: Controle de espécies associadas a fazendas 
* Controle de motoristas: Cadastro de condutores responsáveis pelo transporte.
* Gestão de Hortifruits: Cadastro de pontos de destino das cargas
* Registo de envios: Registro logístico integrando origem, destino, motorista, produto, quantidade e data automática.

# Tecnologias 
Linguagem: Python 3 
Banco de Dados: My SQL (server/workbench)
Biblioteca : 'mysql-connector-python'
